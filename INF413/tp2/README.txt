This python based software implements a k-means algorithm. Data must be written in a file named 'data.txt' as follow: coord1, coord2, ..., coordn, string\ncoord1', ...
It will be asked to you how many groups you want to create and what norm calculation you want to use. It is possible to choose between different types of norm calculation - Euclidean, Manhattan and Sup are currently available.
The resulted groups will be written in a file names 'results.txt' as follow : number of the group, center of the group, list of the points of the group and their properties.
This software is designed for python 3 but should also work with python 2.


Ce programme en python implémente un algorithem des k-means. Les données en entrée doivent être écrites dans un fichier nommé 'data.txt' comme suit : coordonnée1, coordonnée2, ..., coordonéen, chaîne de caractères\ncoordonnée1, ...
Il vous sera demandé de choisir le nombre de groupes que vous voulez créer ainsi que la norme à utiliser. Il est possible de choisir différents types de normes (les normes euclidiennes, de Manhattan et sup' sont actuellement disponibles).
Les résultats seront écrits dans un fichier nommé 'results.txt' comme suit : numéro de groupe, centre du groupe, liste des points du groupe et de leurs propriétés.

Ce programme est conçu pour python 3 mais devrait aussi fonctionner sous python 2.
